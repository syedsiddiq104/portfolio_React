# Personal Portfolio Website

This is my personal portfolio website built using React and Vite.  
It showcases my projects, skills, and contact information.

## Live Demo
https://portfolio-react-ti2f.vercel.app

## Tech Stack
- React
- JavaScript
- HTML5
- CSS3
- Vite

## Features
- Responsive design
- Animated UI
- Projects showcase
- Contact section
- Smooth navigation

## Installation

Clone the repository

git clone https://github.com/syedsiddiq104/portfolio_React.git

Navigate into the project

cd portfolio_React

Install dependencies

npm install

Run the development server

npm run dev

## Deployment
This project is deployed using Vercel.
